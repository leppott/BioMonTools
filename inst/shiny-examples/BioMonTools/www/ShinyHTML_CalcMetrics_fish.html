Purpose

This function calculates an assortment of biological metrics based on the traits/attributes in the input file.

Click here to see the list of available metrics for fish.

You can request the addition of new metrics to the R package or Shiny app by emailing Erik (Erik.Leppo@tetratech.com) or posting your request on the BioMonTools GitHub page (https://github.com/leppott/BioMonTools/discussions).
